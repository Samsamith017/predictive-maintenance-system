	Industrial Predictive Maintenance Dashboard:

This project demonstrates a machine learning–based approach to predictive maintenance, where equipment health is monitored and potential failures are identified using real-time operational inputs. The application is built using Streamlit and provides a simple, interactive way to simulate industrial machine conditions.

Live Demo
The application can be accessed directly through the following link:
	https://equipment-reliability-dashboard.streamlit.app/

Repository
The source code for this project is available at:

Overview
In many industrial settings, unexpected machine failures can lead to downtime and increased operational costs. Predictive maintenance aims to address this by identifying warning signs early, allowing for timely intervention.

This project reflects that idea in a simplified form. By entering key machine parameters, users can observe how a trained model evaluates the condition of the system and estimates the associated risk level.

Functionality

The application allows users to input the following parameters:

	💠Air temperature
	💠Process temperature
	💠Rotational speed
	💠Torque
	💠Tool wear
	💠Machine type (Low, Medium, High)
Based on these inputs, the model analyzes the data and produces a probability-based assessment of machine condition. This is translated into a risk score and categorized as:

Low risk
Medium risk
High risk

A confidence breakdown is also displayed to give a clearer understanding of the prediction.
